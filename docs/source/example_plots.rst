Example Results and Plotting
============================

`Example-Resuts/` contains example result files and example plotting scripts.

Bulk example
------------

.. code-block:: bash

   python Example-Resuts/Bulk/plot_bulk.py

Output:

- `Example-Resuts/Bulk/Comparison_NEP_vs_DFT.pdf`

Film example
------------

.. code-block:: bash

   python Example-Resuts/Film-1nm/plot_film.py

Output:

- `Example-Resuts/Film-1nm/Si_film_1nm_4panel.pdf`

These plotting scripts are examples and can be copied/modified for your own data.
